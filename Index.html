Born of Putridy

Historia do Jogo

O jogo contara a historia de um personagem que perdeu sua mão quando nasceu, um dia seu irmão mais velho recebe uma chamada urgente de uma parte da sua familia mais distante logo decide viajar para averigar a situação desta parte mais remota da familia onde desaparece completamente deixando apenas uma carta que tem como destinatario seu irmão mais novo que por sua vez a recebe, desesperados o nosso personagem e seu pai ligam para autoridades locais para que haja uma averiguação do local, apos a investigação seu irmão e dado como desaparecido. anos se passam agora ja adulto, decide que tera como objetivo principal a investifação do seu irmão desaparecido, infelismente seu pai não lidou tão bem com isso e acabou se afundando em drogas e bebidas oque ocasionol em seu declineo mental que teve seu final em um suicidio

Items

o jogo contara com 30 items divididos entre utilizaveis,consumiveis,armas e as chamadas Reliquias

Tecnologias

até o momento estou utilizando 3 tecnologias sendo elas

    React

    "O React é uma biblioteca front-end JavaScript de código aberto com foco em criar interfaces de usuário em páginas web" React esta sendo usado para a contrução do projeto sendo ela a tecnologia mais usada utilizada para criar o projeto particulamente todo em si.

    Redux

    "Redux é uma biblioteca JavaScript de código aberto para gerenciar o estado do aplicativo" Redux esta sendo utilizado para a criação de funções e interaçoes com o inventario alem de ser feito por tal.

    CSS

    "Cascading Style Sheets é um mecanismo para adicionar estilos a uma página web, aplicado diretamente nas tags HTML ou ficar contido dentro das tags de styleTambém é possível, adicionar estilos adicionando um link para um arquivo CSS que contém os estilos." esta sendo utilizada para estilização do site como localização do iventario,cor do fundo animações e etc.

    Canva

    "Canva é uma plataforma de design gráfico que permite aos usuários criar gráficos de mídia social, apresentações, infográficos, pôsteres e outros conteúdos visuais" Canva esta sendo utilizado para a criação da logo dos items em geral
